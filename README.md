# Proyecto Final Dijitalización de Plantas Productivaas
Esta aplicación funciona conectandose con una base de datos de InfluxDB, de allí se obtienen las mediciones de un sensor de temperatura y humedad y otro sensor de vibraciones. Estos datos luego se procesan y se entrega el máximo, minimo y promedio del período consultado.

La aplicación también tiene una función de predicción de datos futuros usando el modelo Prophet, que permite elejir cuanto tiempo a futuro predecir de cada variable.
